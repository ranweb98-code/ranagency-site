# RAN Agency - Website Vision

# Goal

This website should not feel like a normal agency website.

It should create a "wow" moment within the first few seconds.

The experience should communicate:

- Premium
- Luxury
- Innovation
- AI
- High-end branding
- Confidence
- Motion
- Modern design

Every section should feel alive.

Nothing should feel static.


---

# Overall Philosophy

This is NOT another template website.

It should feel handcrafted.

Every scroll should reveal something new.

Every interaction should feel intentional.

The visitor should immediately understand that if this is the quality of the website, then the services must be even better.

The goal is to make visitors think:

"I need these people to build my business."

---

# Motion Design

Motion is one of the highest priorities.

Almost everything should have subtle movement.

Examples:

- floating elements
- parallax
- smooth reveals
- blur transitions
- depth
- scale animations
- mouse interactions
- magnetic buttons
- cursor reactions
- image movement
- 3D feeling

Animations should feel expensive.

Never flashy.

Never cheap.

Everything should be smooth.

---

# Hero Section

The Hero section should become the signature of the website.

It needs to immediately create a WOW effect.

The current hero image MUST stay exactly as it is.

DO NOT regenerate it.

DO NOT replace it.

DO NOT reduce quality.

DO NOT change the person.

Use the existing image only.

Ideas:

- layered floating images
- stacked image effect
- depth
- glass panels
- floating UI cards
- animated lighting
- mouse parallax
- cinematic movement

I also want to create AI-generated layers from the existing hero image.

Examples:

- separate eyes
- separate head
- separate shoulders
- foreground
- background

When moving the mouse:

- the eyes can subtly follow
- the head can slightly rotate
- different layers move at different speeds

The effect should remain subtle.

Never cartoonish.

On mobile:

Instead of mouse interaction,

the animation should automatically loop slowly forever.

The hero should always feel alive.

---

# Scrolling Experience

Scrolling should feel cinematic.

Ideas:

- smooth transitions
- sections blending together
- images moving independently
- text revealing naturally
- parallax backgrounds
- scaling effects
- floating gradients

Every section should have motion.

---

# Sections

Every section should feel unique.

Avoid repeating layouts.

Alternate between:

- left/right
- centered
- overlapping cards
- full-width visuals
- split layouts
- floating content

The visitor should never feel bored.

---

# Typography

Large.

Bold.

Minimal.

Modern.

Strong hierarchy.

Whitespace is extremely important.

---

# Colors

Dark premium palette.

Modern gradients.

Soft lighting.

Glass effects.

Subtle glows.

Never overuse colors.

---

# Buttons

Buttons should feel premium.

Ideas:

- magnetic hover
- glow
- subtle movement
- animated borders
- micro interactions

---

# Cards

Cards should not simply appear.

They should:

- float
- tilt
- react
- animate
- create depth

---

# Performance

Animations should remain smooth.

Target:

60 FPS

Use GPU accelerated animations.

Avoid unnecessary JavaScript.

Prioritize CSS transforms.

---

# Mobile Experience

Mobile should NOT simply be a smaller desktop.

It should be redesigned where necessary.

Animations should continue automatically.

Interactions should adapt naturally.

Performance is critical.

---

# Inspiration

The quality should be comparable to websites such as:

- Apple
- Stripe
- Linear
- Framer
- Vercel
- Arc Browser
- Nothing
- Raycast

Not copied.

Only inspired by their level of polish.

---

# Final Goal

Create one of the most impressive AI agency websites on the internet.

Every visitor should remember it.

Every interaction should communicate quality.

The website itself should become part of the portfolio.
